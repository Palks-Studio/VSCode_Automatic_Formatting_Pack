VSCode_Pack_Formatage_Automatique/
│
├── .vscode/
│   └── extensions.json                 ← Overrides locaux des extensions (ex : désactiver Prettier, RunOnSave)
│
├── settings.json                       ← Réglages complets de VS Code (indentation, LF, UTF-8, interface éditeur)
├── keybindings.json                    ← Raccourcis : Alt + M (minimap), Alt + R (réindenter le fichier)
├── tasks.json                          ← Tâches de formatage (Autopep8 + script Clean)
├── launch.json                         ← Exécution rapide du fichier Python actif
│
├── clean.py                            ← Nettoyage global (espaces, guillemets, normalisation)
├── convert.py                          ← Conversion CRLF → LF dans tout l’espace de travail
├── space.py                            ← Détection des espaces ou lignes vides superflues
│
├── INSTALL.md                          ← Guide d’installation et d’utilisation
├── README_COMMERCIAL.md                ← Présentation du produit (version Gumroad)
│
└── docs/
    ├── README_TECHNIQUE.md             ← Notes techniques, maintenance, versions futures
    │
    ├── demo.py                         ← Exemple Python cassé (avant) + correction automatique (vidéo)
    ├── demo.js                         ← Code JavaScript désordonné + version formatée
    ├── demo.html                       ← HTML mal indenté + structure corrigée
    ├── demo.css                        ← CSS déformé + sortie normalisée
    ├── demo.json                       ← JSON mal aligné + correction automatique
    │
    └── exemples/
        ├── avant/                      ← Exemples non formatés / cassés
        └── apres/                      ← Résultats propres / corrigés générés par le pack
