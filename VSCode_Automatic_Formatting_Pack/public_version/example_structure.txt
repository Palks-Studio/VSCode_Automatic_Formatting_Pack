VSCode_Automatic_Formatting_Pack/
│
├── .vscode/
│   └── extensions.json                 ← Local extension overrides (e.g., disable Prettier, RunOnSave)
│
├── settings.json                       ← Full VS Code settings (indentation, LF, UTF-8, editor layout)
├── keybindings.json                    ← Shortcuts: Alt + M (minimap), Alt + R (reindent file)
├── tasks.json                          ← Formatting tasks (Autopep8 + Clean script)
├── launch.json                         ← Quick run of the active Python file
│
├── clean.py                            ← Global cleanup (spaces, quotes, normalization)
├── convert.py                          ← CRLF → LF conversion across the workspace
├── space.py                            ← Detection of extra spaces or empty lines
│
├── INSTALL.md                          ← Installation and usage guide
├── COMMERCIAL_README.md                ← Gumroad product presentation
│
└── docs/
    ├── TECHNICAL_README.md             ← Technical notes, maintenance, future versions
    │
    ├── demo.py                         ← Python broken file (before) + auto-fix (video example)
    ├── demo.js                         ← JavaScript messy file + formatted version
    ├── demo.html                       ← HTML badly indented + corrected structure
    ├── demo.css                        ← CSS deformed + normalized output
    ├── demo.json                       ← JSON misaligned + auto-corrected
    │
    └── examples/
        ├── before/                      ← Unformatted / broken examples
        └── after/                      ← Clean / fixed results generated by the pack
